    Propiedades del flexbox
  • flex-direction: define la direccion que se colocara los elementos dentro del contenedor
  • Justify-content: define la alineacion horizontal de los elementos flex dentro del contenedor
  • align-item: define la alineacion vertical de los elementos flex dentro del contenedor
  • flex-wrap: define los elementos flex cuando deben envolverse o no cuando el ancho del contenedor no es suficiente
  • align-content: define los elementos flex en caso de se hayan envuelto en varias lineas
    Propiedades del flex
  • flex-grow: define los espacios adicionales a un elemento de flex flex-direction
  • flex-shrink: define los espacios que debe de reducir cuando se reduzcan los contenedores
  • flex-basic: la propiedad que combina las tres propiedades
